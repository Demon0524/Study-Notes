### 自定义bridge网络
> 优点：实现DNS解析、与其他网络隔离
```perl
# docker network create --driver [bridge] [net_name]

[node2@Serverb ~]$ docker network create --driver bridge bridge_test
a1c8f2a86d0f165d2386a4f7b2ff8c602b85de00575f60c10676e621fd185628
[node2@Serverb ~]$ docker network ls
NETWORK ID     NAME          DRIVER    SCOPE
da745a8f1791   bridge        bridge    local
a1c8f2a86d0f   bridge_test   bridge    local
e36c702a1b18   host          host      local
47e34eef3822   none          null      local
```
